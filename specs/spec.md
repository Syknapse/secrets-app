# Secrets sharing app

Create an app for secrets sharing using React + TS.
The web app should show a form with an input. Users can input a secret and generate a link to safely share with others.
The link should be safe and secure and work wherever the user opens it.
It should have an extra feature that allows a user to optionally select that the link auto-destroys after a single viewing, or set a timestamp as the expiry date.
